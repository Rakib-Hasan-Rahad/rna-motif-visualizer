🚀 QUICK START INSTRUCTIONS
================================================================================

Installation (3 steps):

1. Copy plugin folder to PyMOL:
   
   macOS/Linux:
   cp -r rna_motif_visualizer ~/.pymol/startup/
   
   Windows:
   Copy rna_motif_visualizer to:
   C:\Users\<YourName>\AppData\Roaming\PyMOL\startup\

2. Restart PyMOL
   
   Close and reopen PyMOL. You should see:
   "RNA MOTIF VISUALIZER v1.0.0"
   in the console.

3. Start using:
   
   rna_load 1S72
   rna_status
   rna_toggle KTURN on

Usage (3 commands):

   rna_load <PDB_ID_or_PATH>    # Load structure
   rna_toggle <MOTIF> <on|off>  # Toggle visibility
   rna_status                    # Show information

Examples:

   rna_load 1S72                # Load ribosomal RNA
   rna_toggle KTURN on          # Show K-turns in red
   rna_toggle GNRA on           # Show GNRA in yellow
   rna_status                   # Show loaded motifs
